# 🎵 基于 Vue 3 + Typescript + Vite + Pinia + Element-plus 的高仿网易云mac客户端播放器（PC）

------

参考了很多个github的仿网易云开源项目一步一步自己搭的，也是第一次自己写一个音乐播放器，用到的新技术还是很多的，有喜欢的可以一起学习，欢迎提出建议和star✨✨✨
